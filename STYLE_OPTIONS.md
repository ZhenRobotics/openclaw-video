---

## 方案A：纯文字阴影 💎

### 优点
- ✅ **简洁优雅** - 无多余元素，商务感强
- ✅ **不遮挡背景** - 100%显示背景视频内容
- ✅ **高级感** - 类似Apple/Tesla发布会风格
- ✅ **文件较小** - 27 MB

### 缺点
- ⚠️ 复杂背景下可读性稍弱（但已增强双层阴影）

### 技术实现
```tsx
// 双层阴影系统
textShadow: '0 3px 12px rgba(0, 0, 0, 0.8), 0 1px 3px rgba(0, 0, 0, 0.9)'
subtitleShadow: '0 2px 8px rgba(0, 0, 0, 0.75), 0 1px 2px rgba(0, 0, 0, 0.85)'
```

### 适用场景
- 高端商务宣传片
- 品牌形象视频
- 产品发布会
- 医疗科普（当前）

---

## 方案B：半透明背景框 📺

### 优点
- ✅ **可读性最强** - 100%清晰，任何背景都完美
- ✅ **专业字幕感** - 类似电视字幕、在线课程风格
- ✅ **稳定一致** - 不受背景变化影响

### 缺点
- ⚠️ 遮挡部分背景视频（约30%文字区域）
- ⚠️ 略显"工具感"，高级感稍弱

### 技术实现
```tsx
// 半透明背景框
backgroundColor: 'rgba(0, 0, 0, 0.6)',  // 60%不透明黑色
padding: '40px 50px',
borderRadius: '12px',
backdropFilter: 'blur(10px)',  // 高斯模糊
boxShadow: '0 8px 32px rgba(0, 0, 0, 0.5)',  // 背景框阴影
```

### 适用场景
- 教学视频/课程
- 新闻播报风格
- 字幕较长的内容
- 复杂背景（快速运动、高对比度）

## 🎬 切换方法

### 快速切换命令

```bash
# 切换到方案A（纯文字阴影）
./scripts/switch-style.sh text-shadow
npm run build

# 切换到方案B（半透明背景框）
./scripts/switch-style.sh background-box
npm run build
```

### 手动切换

```bash
# 方案A：纯文字阴影
cp src/premium-scenes-text-shadow.tsx src/premium-scenes.tsx
npm run build

# 方案B：半透明背景框（当前）
# 已经是默认版本
npm run build
```

## 🎯 推荐选择

### 脑卒中康复视频 → **方案A（纯文字阴影）**

**理由**：
1. 医疗健康类内容需要**专业、权威、高端**的视觉风格
2. 背景视频展示康复训练场景，需要**完整可见**
3. 当前双层阴影已确保可读性
4. 符合医疗宣传片的行业标准

### 其他场景建议

- **企业宣传片** → 方案A
- **产品发布会** → 方案A
- **在线教育课程** → 方案B
- **新闻资讯** → 方案B
- **字幕密集内容** → 方案B
