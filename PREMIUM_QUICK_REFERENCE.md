## 代码示例速查

### 最简单的用法

```javascript
// scenes-data.ts

export const scenes = [
  // Elegance - 优雅开场
  {
    start: 0,
    end: 5,
    type: 'elegance',
    styleTheme: 'premium',
    title: '全新旗舰产品',
    subtitle: '即将发布',
  },

  // Authority - 数据展示
  {
    start: 5,
    end: 12,
    type: 'authority',
    styleTheme: 'premium',
    title: '营收增长',
    number: '340%',
    subtitle: '连续三年三位数增长',
  },

  // Cinematic - 震撼收尾
  {
    start: 12,
    end: 18,
    type: 'cinematic',
    styleTheme: 'premium',
    title: '重新定义未来',
    subtitle: '敬请期待',
  },
];
```

### 混合风格示例

```javascript
export const scenes = [
  // Premium 开场
  { type: 'elegance', styleTheme: 'premium', title: '企业介绍' },

  // Cyber 中场（突出科技感）
  { type: 'emphasis', styleTheme: 'cyber', title: 'AI 驱动' },

  // Premium 收尾
  { type: 'authority', styleTheme: 'premium', number: '500%' },
];
```

  --input script.txt \
  --style-theme premium \

## 故障排查

### 问题: 文字看不清

**Cyber 风格**:
- 调整 `bgOpacity`（降低背景视频透明度）
- 增加 `textShadow` 强度

**Premium 风格**:
- 使用更深的背景色
- 增加文字粗细（font-weight）

### 问题: 动画太慢/太快

**调整动画时长**:
```javascript
// Elegance: 默认 2 秒
const titleOpacity = interpolate(frame, [0, 60], [0, 1], ...);
                                        // ^^^ 改为 30（1 秒）或 90（3 秒）

// Authority: 默认 1.5 秒
const titleTranslateX = interpolate(frame, [0, 45], [-50, 0], ...);
                                           // ^^^ 调整第二个值
```

### 问题: 特效太多/太少

**Cyber 风格**:
- 关闭粒子爆炸: 修改 `useParticles = false`
- 减少粒子数量: `particleCount = 25`（默认 50）

**Premium 风格**:
- 已经最简化，建议不要增加特效
