### 1. 扫光特效（LightSweep）⚡⭐⭐⭐⭐⭐

**功能**: 高潮场景（title 和 emphasis）添加从左至右的扫光效果

**实现位置**: `src/SceneRenderer.tsx:417-463`

**技术细节**:
```typescript
const LightSweep: React.FC<{ frame: number; duration?: number }> = ({
  frame,
  duration = 60, // 2 秒扫过
})
```

**关键参数**:
- **扫光位置**: -100% → 200%（完全扫过屏幕）
- **透明度变化**: 0 → 0.8 → 0.8 → 0（中间最亮）
- **宽度**: 40% 屏幕宽度
- **颜色渐变**:
  ```
  Cyan 0% → Cyan 30% → White 50% → Magenta 70% → Magenta 100%
  ```
- **模糊**: 20px（柔和光效）
- **混合模式**: `screen`（光效叠加）

**缓动函数**: `easeInOutCubic` (0.25, 0.1, 0.25, 1)

**视觉效果**:
- 从左至右平滑扫过
- 中间白色最亮，两端青色/洋红
- 模糊+screen 混合产生电影级光效
- 不干扰文字阅读

**应用场景**:
- ✅ title 场景
- ✅ emphasis 场景
- ❌ 其他场景（保持简洁）

### 2. 镜头运动效果（Camera Movement）🎬⭐⭐⭐⭐⭐

**功能**: 为不同场景添加专业级镜头推拉和晃动效果

**实现位置**: `src/SceneRenderer.tsx:488-533`

**技术细节**:
```typescript
const getCameraEffect = (frame: number, sceneType: string) => {
  // 返回 scale, translateX, translateY
}
```

**不同场景的镜头运动**:

#### Title 场景 - 推进镜头
```typescript
scale: 1.2 → 1 (从远拉近)
translateY: -30px → 0 (从上方推进)
duration: 40 帧 (1.33 秒)
easing: easeOutCubic
```

**效果**: 标题从远处逐渐推进到观众眼前，增加开场冲击力

#### Emphasis 场景 - 震动效果
```typescript
translateX: ±2px (左右晃动)
translateY: ±3px (上下晃动)
frequency: sin(frame * 0.8)
decay: max(0, 1 - frame / 20) (逐渐减弱)
```

**效果**: 强调内容时轻微震动，吸引注意力，但不影响阅读

#### End 场景 - 拉远镜头
```typescript
scale: 1 → 0.9 (拉远)
duration: 30 帧 (1 秒)
easing: easeInQuad
```

**效果**: 结尾时镜头拉远，给观众"离开"的感觉

#### 其他场景
```typescript
scale: 1, translateX: 0, translateY: 0
```

**效果**: 保持静态，不干扰内容阅读

**应用方式**:
- 整个场景包裹在镜头运动 wrapper 中
- 所有元素（背景、文字、粒子、扫光）都受影响
- 创造统一的镜头运动感

### 3. 3D 文字旋转优化 🔄⭐⭐⭐⭐

**问题**: 之前无限旋转，观众无法聚焦

**解决方案**: 三段式旋转 + 停顿 + 动态光效

**实现位置**: `src/SceneRenderer.tsx:251-328`

**三段式旋转**:
```typescript
// 帧数:   0      30        60        90
// 角度:   0°  →  270°  →  270°  →  360°
//         快速旋转   停顿展示   慢速完成
```

**关键优化**:

1. **动态景深调整**
   ```typescript
   perspective: 1500px → 800px (拉近镜头)
   ```

2. **动态缩放**
   ```typescript
   scale: 0.8 → 1.2 → 1 (旋转时放大，停顿时收缩)
   ```

3. **动态光效强度**
   ```typescript
   // 根据旋转角度调整光效
   glowIntensity: 0.3 → 0.8 → 1.0

   // 正面时最亮（270度时）
   rotationY: Math.PI * 1.5 → glowIntensity: 1.0
   ```

4. **增强阴影**
   ```typescript
   textShadow: 动态调整 (2px → 8px)
   filter: drop-shadow(0 0 ${20 * glowIntensity}px)
   ```

**视觉效果**:
- 快速旋转引起注意
- 停顿在最佳角度（270度）让观众聚焦
- 光效在正面时最强，增强视觉冲击
- 最后慢速完成，过渡自然

**缓动函数**: `easeInOutCubic` (0.65, 0, 0.35, 1)
