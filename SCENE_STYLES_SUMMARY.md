---

## 📊 场景风格统计

**总数**: 6种场景风格

| # | 风格类型 | 英文名 | 主要特效 | 适用场景 |
|---|---------|--------|---------|---------|
| 1 | 标题场景 | `title` | 故障效果 + 弹性缩放 | 视频开头，介绍主题 |
| 2 | 强调场景 | `emphasis` | 猛击放大 + 快速弹出 | 强调重点，突出核心信息 |
| 3 | 痛点场景 | `pain` | 震动抖动效果 | 描述问题，表达痛点 |
| 4 | 圆圈高亮 | `circle` | 圆圈绘制 + 爪子指示 | 高亮特定内容，列表展示 |
| 5 | 常规内容 | `content` | 平滑缩放 + 淡入 | 通用场景，常规信息 |
| 6 | 结束场景 | `end` | 上升淡出效果 | 视频结尾，行动号召 |

## 🎨 详细特效说明

### 1. 标题场景 (title)

**视觉效果**:
- ✨ 故障效果 (Glitch)：文字带彩色偏移阴影
- 🎯 弹性缩放：从0到1弹跳式放大
- 💫 镜头推进：从远到近的推镜效果

**技术参数**:
```javascript
- Spring: bouncy (弹性强)
- 故障量: sin(frame * 0.5) * 3
- 淡入: 0→1 (10帧)
- 推镜: scale 0.8→1.0
```

**适用文本**: 
- 产品名称
- 主标题
- 章节标题

---

### 2. 强调场景 (emphasis)

**视觉效果**:
- 💥 猛击效果 (Slam)：快速从0.5放大到1.1
- ⚡ 爆炸式出现：突然且有力
- 🎆 震撼感强：吸引注意力

**技术参数**:
```javascript
- Spring: snappy (快速响应)
- 缩放: 0.5→1.1
- 震动: sin(frame * 0.8) * 3px
- 3D旋转: 270°暂停后到360°
```

**适用文本**:
- 核心卖点
- 重要数据
- 关键结论
- 惊人效果

---

### 3. 痛点场景 (pain)

**视觉效果**:
- 😰 震动抖动：表达不安/痛苦
- 📉 负面情绪：传达问题严重性
- ⚠️ 警示效果：引起重视

**技术参数**:
```javascript
- 抖动: sin(frame * 0.8) * 2px
- 持续抖动: 整个场景
- 颜色: 可配置为红色/橙色
```

**适用文本**:
- 痛点描述
- 问题场景
- 困难挑战
- 负面案例

---

### 4. 圆圈高亮 (circle)

**视觉效果**:
- ⭕ 圆圈绘制：逐渐画出圆形边框
- 🐾 爪子指示：小猫爪子指向重点
- 🎯 精确聚焦：引导视线

**技术参数**:
```javascript
- 淡入: 0→1 (15帧, cubic easing)
- 圆圈动画: 0→360° 绘制
- 爪子动画: 同步出现
```

**适用文本**:
- 列表项
- 关键词
- 特定数字
- 需要标注的内容

---

### 5. 常规内容 (content)

**视觉效果**:
- 📄 平滑过渡：专业流畅
- 🎬 电影级缓动：cubic bezier
- 💼 商务风格：稳重大气

**技术参数**:
```javascript
- 缩放: 0.95→1.0 (20帧)
- 淡入: 0→1 (15帧)
- Easing: easeOutCubic (专业级)
- 色彩呼吸: 85%↔115% 亮度
```

**适用文本**:
- 通用信息
- 功能介绍
- 详细说明
- 过渡内容

---

### 6. 结束场景 (end)

**视觉效果**:
- 🚀 上升消失：向上飘走
- 👋 告别动画：温和结束
- 🌅 淡出效果：优雅离场

**技术参数**:
```javascript
- 上升: 0→-50px (30帧)
- 淡出: 1→0 (10帧, 从第20帧开始)
- Easing: easeInCubic
- 拉远镜头: scale 1.0→0.9
```

**适用文本**:
- 行动号召 (CTA)
- 联系方式
- 感谢观看
- 下期预告

## 🎭 小猫吉祥物动作

每个场景还可以配置小猫(Xiaomo)的动作：

| 动作 | 英文名 | 说明 | 适合场景 |
|------|--------|------|---------|
| 偷看 | `peek` | 从角落偷看 | title, 开场 |
| 躺卧 | `lie` | 趴着休息 | content |

| 画圈 | `circle` | 画圆圈 | circle |
| 思考 | `think` | 思考姿势 | content |
| 挥手 | `wave` | 挥手告别 | end |

## 💡 最佳实践建议

### 场景类型选择

1. **视频开头** → `title`
   - 吸引注意
   - 建立主题

2. **核心卖点** → `emphasis`
   - 强调重点
   - 突出优势

3. **问题描述** → `pain`
   - 引起共鸣
   - 表达痛点

4. **列表展示** → `circle`
   - 清晰明了
   - 逐项说明

5. **详细说明** → `content`
   - 平稳过渡
   - 信息传达

6. **视频结尾** → `end`
   - 行动号召
   - 优雅结束

### 视觉节奏控制

**快节奏视频** (15-30秒):
```
title → emphasis → content → emphasis → end
```

**中等节奏** (30-60秒):
```
title → content → pain → emphasis → content → circle → end
```

**慢节奏视频** (60秒+):
```
title → content → pain → content → circle → emphasis → content → end
```

## 🔧 自定义配置

### 场景类型配置
```typescript
{
  type: 'emphasis',           // 场景类型
  title: '核心卖点',
  color: '#FF0066',          // 自定义颜色
  xiaomo: 'point',           // 小猫动作
  bgVideo: 'tech/code.mp4',  // 背景视频
  bgOpacity: 0.3             // 背景透明度
```

